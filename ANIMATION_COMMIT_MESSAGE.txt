feat: Add smooth page transition animations to Student Dashboard

Implemented CSS-based page entrance animations across all 5 Student Dashboard 
pages for a polished and professional user experience.

## Animation Features
- Page fade-in: Opacity transition from 0 to 1 over 0.5s
- Slide-up motion: Transform translateY from 20px to 0 over 0.5s
- Staggered list animations: Quiz cards slide in with 0.1s delays (StudentDashboard)
- Timing function: ease-out for smooth, natural motion
- Trigger: Automated 50ms after component mount via pageLoaded state

## Implementation Details
Each page received the following updates:
1. Added `pageLoaded` state variable (useState)
2. Added animation trigger in useEffect: `setTimeout(() => setPageLoaded(true), 50)`
3. Updated main container CSS with conditional opacity/transform properties
4. Applied transitions to respect existing dark mode system

## Technical Approach
- Pure CSS transitions (no animation libraries)
- Hardware-accelerated properties (opacity, transform)
- Non-blocking animations that don't affect page functionality
- Consistent with existing inline style patterns in codebase
- Compatible with dark mode toggle transitions
